开发语言: `GO 1.12.1` \
开发工具: `VSCode` \

### Installation
```bash

$ cd go_line_chat
$ ./Build.sh

```
>
> 将生成可执行文件到go_line_chat/Bin下, 包括mac/windows/linux程序,现只有windows可以看到图标
>
